# QuickSpace Landing Page

A responsive marketing landing page built with HTML and CSS. The page showcases the QuickSpace product with a hero section, feature highlights, testimonials, pricing plans, and a newsletter signup.

## Features

- **Responsive navigation** — Desktop nav links and a mobile sidebar menu (CSS-only toggle)
- **Hero section** — Headline, call-to-action buttons, and decorative notification cards
- **Perfect Solution** — Three feature cards with icons and CTAs
- **Discover & Explore** — Dashboard preview with supporting graphics
- **Customer testimonials** — Quote cards with client profiles
- **Pricing** — Billing toggle (visual only) and three plan tiers (Base, Standard, Unlimited)
- **Newsletter** — Email signup form
- **Footer** — Contact info, links, and social icons

## Tech Stack

- HTML5
- CSS3 (modular stylesheets per section)
- [Google Fonts — Poppins](https://fonts.google.com/specimen/Poppins)

## Getting Started

No build step or dependencies required.

### Option 1: Open directly

Double-click `index.html`, or run:

```bash
open index.html
```

### Option 2: Live Server (recommended for development)

1. Install the **Live Server** extension in Cursor / VS Code
2. Right-click `index.html` → **Open with Live Server**

The page reloads automatically when you save changes.
